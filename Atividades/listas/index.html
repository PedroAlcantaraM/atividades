Listas

O intuito deste exercício é reforçar o aprendizado sobre Listas utilizando também as tags já aprendidas anteriormente.

Itens e Sub-itens com a tag Ul

É possivel "aninhar" listas dentro de outras, vejamos:

  • Item 1
  • Item 2
    • Item 2.1
    • Item 2.2
    • Item 2.3
      • Item 2.3.1
      • Item 2.3.2
    • Item 2.4
  • Item 3

Itens e Sub-itens com a tag Ol

Assim como visto no parágrafo anterior, também é possivel aninhar listas com a tag Ol:

  1. Item 1
  2. Item 2
    1. Item 2.1
    2. Item 2.2
    3. Item 2.3
  3. Item 3
